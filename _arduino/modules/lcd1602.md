# LCD1602

## Подключение {#podklyuchenie}

Подключается через отдельный I2C модуль

| I2C модуль | ARDUINO |
| :--------: | :-----: |
| SDA        | SDA     |
| SDL        | SDL     |
| GND        | GND     |
| VCC        | 5V      |
